React Vite chess client

Quick start:
- npm install
- npm run dev

Notes:
- Put piece images under `public/assets/pieces/` named: wp.png wr.png wn.png wb.png wq.png wk.png and bp.png br.png bn.png bb.png bq.png bk.png
- Configure backend base URL with `VITE_API_BASE` / `VITE_WS_BASE` env or edit `src/config.js`.
